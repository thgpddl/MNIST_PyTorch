# MNIST_PyTorch
pytorch实现的MNIST

### 文件释义
- data.py：loader生成器，会从Torch云端下载MNIST数据集，并处理成train_loader和test_loader返回
- data_vis.py：MNIST数据集可视化，会显示60张MNIST图片
- model.py：定义模型结构的文件
- train.py：网络训练
- inference.py：网络推理
